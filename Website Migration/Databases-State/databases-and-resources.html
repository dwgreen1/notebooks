These resources are all available to residents and employees of the State of Ohio.  

Residents can access all of these resources in the library, and can access some of the resources online.  Resources under the "State of Ohio Employees" section can be accessed by Ohio residents in the library.

State Employees can access all of these resources in the library or remotely using either their myOhio OH|ID or with a State Library of Ohio state employee library card.

 

Ohio Residents

The databases and resources listed below are available for State of Ohio residents to access remotely with a State Library of Ohio library card. Ohio residents can get a library card here.

The resources in the "Employees" section are also available to Ohio residents, but Ohio residents must be on the premises of the State Library to gain access.
